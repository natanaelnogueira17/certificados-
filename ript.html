        O que é ECMAScript 6?
        ECMAScript 6, ou ECMAScript 2015, como também é conhecido, é tão somente a padronização da linguagem JavaScript que conhecemos há tanto tempo.
        De onde surgiu o ECMAScript?
        Todas as linguagem de programação do mundo que se popularizam amplamente, precisam de uma padronização, que é feita por um órgão responsável. Esta padronização ajuda a linguagem a ser utilizada da maneira correta, a manter-se dentro de um esquema e ajuda, ainda, novos estudantes de desenvolvimento a terem acesso aos seus conceitos de maneira mais compreensível. Para padronizar o JavaScript, a Netscape o enviou para a ECMA Internacional,uma instituição criada na década de 60 para padronizar processos do mercado de tecnologia da informação.
